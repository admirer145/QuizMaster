## Quick Overview

The Quiz Hub allows you to:
- Browse all public quizzes created by the community
- Filter by category, difficulty, and creator
- Add quizzes to your personal library
- View quiz details before starting

## Adding to Your Library

1. Browse quizzes in the Quiz Hub
2. Click **Add to Library** on any quiz
3. The quiz will appear on your Home page
4. Click **Start Quiz** to begin!
